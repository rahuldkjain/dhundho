# Dhundho Backend

## Search Filter

```
$ export FLASK_APP=Search.py
```
```
$ export FLASK_DEBUG=1
```
```
$ flask run
```

## Search Backend

deploy snapshot jar file in search folder

Open Project in Intelli J and Run it

backend - localhost:8080/search/suggestion?query=<query> link to search
 


# Dhundho Frontend
```
$ cd dhundho-frontend
```
```
$ npm install
```
```
$ npm run serve
```
